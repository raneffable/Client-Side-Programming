Welcome to Ran's Game

This is the Introduction of the game.

So this game have genre (MMORPGs) Massively multiplayer online role-playing games.
We have 5 type for the character.

The first is Human.
In Human we have 2 classes which is : Warrior and Mage

The second is Elf.
In Elf we have 2 classes which is : Priest and Archer

The third is Untamed.
In Untamed we have 2 classes which is : Bestial and FoxLady

The fourth is Mermaid.
In Mermaid we have 2 classes which is : Assasin and Psychic.

The fifth is God.
In God we have 2 classes which is : Seeker and Mystic.
